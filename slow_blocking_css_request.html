Slow Blocking CSS Request

This page is slow to load because of a slow CSS file in the header. That's because CSS is render blocking.

Prevent this by ensuring that CSS files are as small and fast to download as possible.

Consider inlining CSS for above the fold markup in the <head> and load an additional CSS file for below the fold content.

Render-blocking is intentional - what good is your site without CSS? (Demo flash of unstyled content - FOUC).
